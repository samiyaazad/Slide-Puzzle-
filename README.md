## Introduction 

I created a slide puzzle using HTML, CSS, and JavaScript. It's DORAEMON, the puzzle. If someone doesn't know who Doraemon is, it a Japanese anime character. 


## Technologies 

I have used : 
             - VS Code
             - HTML
             - CSS
             - JavaScript


## Features 

Not the most updated right now, but you can: 
                                            - Displays a 3x3 sliding puzzle board
                                            - Allows dragging and dropping of tiles
                                            - Only allows valid moves (adjacent tiles)
                                            - Keeps track of the number of moves (turns)
                                            - Detects when the puzzle is solved


## KeyBoard Shortcuts

No, special keyboard shortcuts for now, but definitely gonna add more


## The Process 

I made it with a simple structure, but it took too much time because of figuring out the tiles' placement .
Here is the process: 
                    - Set up the basic HTML structure with a title, game board, and turn counter
                    - Styled the layout using CSS (grid-like board, tile sizes, alignment, colors)
                    - Created a 3x3 board dynamically using JavaScript loops
                    - Assigned each tile an image and unique ID based on its position
                    - Implemented drag-and-drop functionality using event listeners:
                    - dragstart, dragover, dragenter, drop, dragend
                    - Stored the currently dragged tile and the target tile
                    - Added logic to allow movement only if the tiles are adjacent
                    - Swapped tile images to simulate movement
                    - Counted and displayed the number of turns
